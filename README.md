# Tom-Jerry
This game is similar to Tic Tac Toe. It is a two player game, where 1st player is Tom and 2nd Player is Jerry, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a diagonal, horizontal, or vertical row is the winner. It is a solved game with a forced draw assuming best play from both players.
